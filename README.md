#### How it works
+   `file.ext` -> `file copy.ext`

#### Usage
Linux, Windows: `ctrl+D`, macOS: `super+D`

#### Config
`Default.sublime-keymap`
```json
[
    { "keys": ["<custom keys>"], "command": "duplicate_file"}
]
```
